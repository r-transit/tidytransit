# Fallback function to convert strings like 5:02:11 10x slower than [`hhmmss_to_seconds()`](https://r-transit.github.io/tidytransit/reference/hhmmss_to_seconds.md), empty strings are converted to NA

Fallback function to convert strings like 5:02:11 10x slower than
[`hhmmss_to_seconds()`](https://r-transit.github.io/tidytransit/reference/hhmmss_to_seconds.md),
empty strings are converted to NA

## Usage

``` r
hhmmss_to_sec_split(hhmmss_str)
```

## Arguments

- hhmmss_str:

  string
