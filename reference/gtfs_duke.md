# Example GTFS data

Data obtained from
<https://data.trilliumtransit.com/gtfs/duke-nc-us/duke-nc-us.zip>.

## Usage

``` r
gtfs_duke
```

## Format

An object of class `tidygtfs` (inherits from `gtfs`) of length 18.
