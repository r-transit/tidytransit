# Create a text listing the first `max_agencies` agencies of the feed

Create a text listing the first `max_agencies` agencies of the feed

## Usage

``` r
agency_info(gtfs_obj, max_agencies = 3)
```

## Arguments

- gtfs_obj:

  gtfs feed (tidygtfs object)

- max_agencies:

  max number of agencies to list before using "..."

## Value

called for side effects
