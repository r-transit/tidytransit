# GTFS feed summary

GTFS feed summary

## Usage

``` r
# S3 method for class 'tidygtfs'
summary(object, ...)
```

## Arguments

- object:

  a tidygtfs object as read by
  [`read_gtfs()`](https://r-transit.github.io/tidytransit/reference/read_gtfs.md)

- ...:

  ignored for tidygtfs

## Value

the tidygtfs object, invisibly
